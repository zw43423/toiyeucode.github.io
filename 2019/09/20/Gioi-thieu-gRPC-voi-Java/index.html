Giới thiệu gRPC với Java

1.Giới thiệu

gRPC là 1 mã nguồn mở RPC Framework được tạo ra bởi Google. gRPC là sự kết hợp giữa Protocol Buffer và HTTP2. Protocol Buffer được phát triển bởi Google và nhẹ hơn, nhanh hơn, cung cấp hiệu năng tốt hơn so với XML hoặc JSON.


2.Tóm tắt

Trong bài viết này mình chỉ hướng dẫn gRPC với IDE IntelliJ và maven.
Bài viết này hướng dẫn các bước sau:
1. Tạo file proto
2. Generate code server và client bằng protocol buffer compiler
3. Hướng dẫn chạy gRPC trên server
4. Hướng dẫn chạy gRPC trên client
